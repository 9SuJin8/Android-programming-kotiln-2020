# Firebase Cloud Messaging
### 요구 사항
* 앱을 시작하면 FCM Token을 액티비티 화면에 표시
* Cloud Messaging에서 알림을 보내면 
  1. 앱이 전면에서 실행중일 때에는 액티비티에 메시지 수신 내용을 표시하고
  2. 앱이 전면에서 실행중이 아닐 때에는 알림을 표시하고 알림 선택시 액티비티가 시작한다.
