# Firebase Authentication

### 요구사항
* LoginActivity에 Sign Up 버튼을 추가하고 버튼을 누르면 새로 계정을 생성한다
* MainActivity에서 Sign Out 버튼을 누르면 로그아웃 하도록 하고, LoginActivity를 시작하게 한다
